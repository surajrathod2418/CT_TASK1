# CODTECH Interactive Quiz Application

This project is a simple interactive quiz application built using **HTML, CSS, and JavaScript**.  
Users can answer multiple-choice questions, receive instant feedback, and view their final score after completing the quiz.

---

## Features

- Multiple-choice questions  
- Instant correct/wrong feedback  
- Score display at the end of the quiz  
- Clean and user-friendly interface  
- Background image for better visual appearance  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## How to Run

1. Download or clone the project files  
2. Open `index.html` in a web browser  
3. Start the quiz and answer the questions  

---

## Output

After completing the quiz, the final score is displayed on the screen.
